# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-05-10

### Added
- Complete project restructuring into a professional modular architecture.
- Environment variable support for personalization.
- New `src/` directory for source code.
- Assets management system.
- Professional documentation (README, CONTRIBUTING, LICENSE, etc.).
- GitHub Action workflows for CI/CD.
- Docker support.
- Pyproject.toml for modern dependency management.

### Changed
- Refactored monolithic `project.py` into multiple specialized modules.
- Improved naming conventions and code readability.
- Decoupled hardcoded personal information from the codebase.

### Fixed
- Improved music loading robustness.
- Fixed potential state transition inconsistencies.
