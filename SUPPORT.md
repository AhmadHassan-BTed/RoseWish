# Support

Thank you for using the Rose Day Wishing Game! If you encounter any issues or have questions, please follow the guidelines below.

## Getting Help

### Documentation
Please check the [docs/](docs/) folder for setup and architecture information. The [README.md](README.md) also contains essential information.

### GitHub Issues
For bugs and feature requests, please use the [GitHub Issues](https://github.com/AhmadHassan-BTed/Rose-Day-Wishing-Game/issues). Please use the provided templates to ensure we have all the necessary information.

### Community Discussions
For general questions, ideas, or to showcase your customized versions of the game, please use [GitHub Discussions](https://github.com/AhmadHassan-BTed/Rose-Day-Wishing-Game/discussions).

## Professional Support
This is an open-source project maintained by volunteers. We do not offer guaranteed response times or paid support. However, we do our best to help the community.
