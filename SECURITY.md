# Security Policy

## Supported Versions

The following versions of the Rose Day Wishing Game are currently being supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

We take the security of our project seriously. If you find a security vulnerability, please do NOT open a public issue. Instead, please report it privately to the maintainers.

You can report security vulnerabilities by:
1. Sending an email to [MAINTAINER_EMAIL]
2. Providing a detailed description of the vulnerability.
3. Including steps to reproduce the issue.

We will acknowledge your report within 48 hours and provide a timeline for a fix.
