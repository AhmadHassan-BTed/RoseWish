# Project Roadmap

This document outlines the planned future features and improvements for the Rose Day Wishing Game.

## Phase 1: Foundation (Completed)
- [x] Initial game logic and character design.
- [x] Basic storyline and dialogues.
- [x] Particle system for petals and snow.
- [x] Professional project restructuring.
- [x] Environment variable support.

## Phase 2: Polish & Experience (Upcoming)
- [ ] Improved character animations (idle, walk, blink).
- [ ] Multiple character skins/outfits selectable via config.
- [ ] Dynamic weather system (rain, heavy snow).
- [ ] Localization support (multiple languages for dialogues).
- [ ] Save/Load system for game progress.

## Phase 3: Advanced Features
- [ ] Mini-games within the world (e.g., collecting flowers).
- [ ] Interactive environment (shaking trees, clicking objects).
- [ ] Sound effect overhaul (step sounds, interaction sounds).
- [ ] Mobile support (using Kivy or similar).

## Phase 4: Community & Ecosystem
- [ ] Modding support for custom stories.
- [ ] Level editor.
- [ ] Online wishing platform integration.
